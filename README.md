# Anime-Image-Colorization

### Introduciton
透過將資料集的圖片轉為灰階，並以原圖為期望成果，來訓練模型，模型是以AutoEncoder為底，先用Encoder壓縮採樣features後，去訓練模型上色，再用Decoder還原。
